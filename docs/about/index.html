About

Wervice is an open and closed source project which builds and designs code in different coding languages everyone can use, fork and edit. But there are also fonts and images on the website and the apps. This media is always free and public. Sometimes there are credits, but sometimes there are no credits. But the often used sources on the website or in the apps are:

- Unsplash (Photos)
- Google Fonts (Fonts)
- Carbon (Code-Photos)
- Pixabay (FX/Images/Videos)
- GitHub (Code Snippets)
