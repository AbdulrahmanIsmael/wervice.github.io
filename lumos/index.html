Lumos Documentation GitHub

Easily setup your own cloud
with Lumos

You can simple and fast setup your own cloud for your team or family with a few simple clicks




Encryption

Lumos features safe AES encryption out of the box to protect your files from hackers.
You can easily encrypt any file in your account with a strong password.


Virus Protection

To prevent viruses from beeing uploaded, the admin can enable virus protection.
This will scan any file uploaded with the ClamAV virus database.


Collaboration

You can easily work together. Lumos has helpful features like a built-in chat with file sharing support This makes collabortation simple and fast.


Administration

Lumos can be easily administrated using the well-organized dashboard. You can simply monitor the system capacity and manage users.
