Lumos Documentation GitHub

Easily setup your own cloud
with Lumos

You can simple and fast setup your own cloud for your team or family with a few simple clicks



Security

Lumos features safe AES encryption out of the box to protect your files from hackers.
You can easily encrypt any file in your account with a strong password. The security advisor helps you do the right decisions.

Virus Protection

To prevent viruses from beeing uploaded, the admin can enable virus protection.
This will scan any file uploaded with the ClamAV virus database.

Collaboration

Lumos makes working together simple and efficient. You can use the built-in chat out-of-the-box and share information and files. If you want to share files with other people, you can create sharing links.

Administration

Lumos can be easily administrated using the well-organized dashboard. You can simply monitor the system capacity and manage users. You can also costumize Lumos.
